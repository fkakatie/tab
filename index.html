Welcome to Tab

A personal new tab with tasks, calendar notes, bookmarks, and a Pomodoro timer. Everything lives in your browser — no account, no sync.

Tasks
Add tasks with due dates and links. Prioritize or park them for later.
Calendar
Add a daily note or reminder to any day.
Bookmarks
Quick-access links, drag to reorder.
Pomodoro
25-minute focus sessions with short and long breaks.

Press ? anytime to see keyboard shortcuts.

Keyboard Shortcuts

N/
New task
B
New bookmark
T
New appointment for today
P
Start / pause pomodoro
R
Reset pomodoro
<
Go back a pomodoro interval
>
Skip to the next pomodoro interval
X
Export data to clipboard
I
Import data from clipboard
?
Show shortcuts
